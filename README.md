# CampusMap (校园地图)

> ⚠️ **项目状态**：目前处于开发调试阶段。**地图地点数据仅供演示，不代表真实地理位置信息**。

CampusMap 是一款基于 SwiftUI 开发的长沙理工大学校园导航应用，旨在为学生提供便捷的地标查询与位置导航服务。

## ✨ 核心功能

- **多校区导航**：支持金盆岭与云塘校区的快速切换。
- **分类过滤**：按教学楼、餐饮、宿舍等分类查看地点。
- **智能计算**：实时计算用户到目标地点的距离及步行预估时间。
- **定位跳转**：列表与地图联动，实现一键定位。

## 🛠️ 技术栈

- **框架**: SwiftUI
- **地图**: MapKit & CoreLocation
- **架构**: MVVM

## 📁 快速查看

- `App/`: 包含主视图 `ContentView` 和业务逻辑 `CampusMapViewModel`。
- `LocationManager/`: 管理地点数据模型及 Mock 数据。

## 🚀 运行环境

- Xcode 15.0+ 
- iOS 17.0+ (推荐使用物理真机或模拟器开启定位模拟)
